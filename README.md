# Real-Time Weather Tracker

A responsive weather forecasting web application built using HTML, CSS, and JavaScript with OpenWeather API integration for real-time weather updates.


## Features

* Real-time weather updates
* City-based weather search
* OpenWeather API integration
* Temperature, humidity, and wind speed display
* Responsive UI design
* Async/Await API handling


## Technologies Used

* HTML5
* CSS3
* JavaScript
* REST API
* OpenWeather API


## Project Objective

The objective of this project is to provide users with real-time weather information dynamically using REST APIs and improve frontend/API integration skills.


## How to Run

1. Download or clone the repository
2. Open `index.html` in browser
3. Enter city name to get weather details
